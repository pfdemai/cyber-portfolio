## Methodology

Every write-up answers these questions:
1. What is the attack technique and how does it work?
2. What did the attacker do, step by step?
3. How would a defender detect this? What would the logs look like?
4. Where does this map in MITRE ATT&CK?

The goal is not to demonstrate that an attack works — it's to document the detection logic.
A SOC analyst needs to know what the attack looks like from the defender's side.
